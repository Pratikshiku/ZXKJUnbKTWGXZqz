## College Student Welcome System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0cc19134b73a48919669d92636560110/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Bulletin Board
5. School Learning
6. Admission Notice
7. Major Introduction
8. Dormitory Information
9. My Favorites


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Student Management
4. Bulletin Board Management
5. School Information Management
6. Admission Notice Management
7. Major Introduction Management
8. Registration Appointment Management
9. Dormitory Information Management
10. Course Information Management
11. Class Information Management
12. System Management (Carousel Management, About Us)
